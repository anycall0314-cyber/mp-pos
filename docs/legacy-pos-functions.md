## 業態啟發摘要

通訊行的營收 / 帳務不是單一條進銷存可以涵蓋的，至少 5 條收入流：

1. **賣手機 / 配件**：傳統進銷存（毛利低）
2. **賣 SIM 卡上線**：與電信商簽促銷方案，賺佣金 + 月結對帳
3. **代收電信費**：代客戶繳費，賺手續費
4. **維修服務**：建單 → 送維修中心 → 報價 → 結案，賺維修費
5. **大量中古**：收購中古機 → 檢測 → 定價 → 再販售

且還有「店內管銷費用」（水電、租金、紙箱、行銷）要記，否則毛利不準。
